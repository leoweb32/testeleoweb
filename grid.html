Grid Layout css

display: grid ou display: inline-grid: Já faz com que todos itens dentro do Container se tornem partes de Grid e se tornem Grid-item.
inline-grid e grid são propriedade que dizem se vai haver ao lado do container um outro bloco alinhado verticalmente ou horizontalmente.
grid-template-columns:Neste itens iremos colocar quantas colunas teremos e quais as suas dimensões.
Exemplo: grid-template-columns: 50px 100px 200px 150px. Com isso temos 4 colunas e cada uma com uma dimensão especificada.
grid-template-rows: É a mesma coisa só que a configuração alterada é a das linhas. (alturas das células)
grid-column-gap e grid-row-gap: Está propriedade serve para os espaçamentos. Se utilizar grid-row-gap você coloca uma propriedade para Rows e Cols
